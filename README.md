# 🚀 Full Stack Developer Portfolio

## ⚙️ Getting Started

### Prerequisites

* Node.js (v16 or higher)
* npm or yarn

### Installation

```bash
git clone https://github.com/ritikattri773/DVA_Portfolio.git
cd DVA_Portfolio
npm install
npm run dev
```
